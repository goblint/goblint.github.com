How to reproduce benchmarking results?

  1. Install & configure Ocaml (≥ 4.00.1), Batteries (≥ 2.0) and Ruby 
    ???
  2. Prepare to build Goblint from Sources 
    see http://goblint.in.tum.de/download.html
  3. Download and prepare the Goblint for Part 1 of benchmarks: 
    git clone -b side-effect2 git://github.com/goblint/analyzer.git analyzer1
cd analyzer1/
make 
./runbench.sh
cd ..
  4. Download and prepare the Goblint for Part 2 of benchmarks: 
    git clone -b box-operator git://github.com/goblint/analyzer.git analyzer2
cd analyzer2/
make 
./runbench.sh
cd ..
