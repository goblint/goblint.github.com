How to reproduce benchmarking results?

  1. Install & configure Ocaml >=3.11 and Ruby 
    ???
  2. Create a directory, where we will install Goblint and some required libraries. 
    mkdir goblint_stuff
cd goblint_stuff
  3. Download and prepare the xml-light library: 
    cvs -d:pserver:anonymous@cvs.motion-twin.com:/cvsroot login
(enter empty password)
cvs -d:pserver:anonymous@cvs.motion-twin.com:/cvsroot co ocaml/xml-light
mv ocaml/xml-light xml-light
rm -rf ocaml
cd xml-light
make
make opt
cd ..
  4. Download and prepare the cil library: 
    git clone git://github.com/kerneis/cil
cd cil
./configure
make
cd ..
  5. Download our benchmarking code: 
    git clone git://github.com/goblint/bench.git
  6. Download and prepare the Goblint (single-threaded version): 
    git clone -n git://github.com/goblint/analyzer.git
cd analyzer/
git checkout side-effect.fun
./configure
make opt
  7. Run & Review single-threaded results: 
    ./scripts/update_bench.rb 
firefox tests/bench_result/index.html &
  8. Download and prepare the Goblint (normal, multi-threaded version): 
    git checkout 367902c8b534a6cc18ccc4affb962353d573b99a
./configure
make opt
  9. Run & Review multi-threaded results: 
    ./scripts/update_bench.rb 
firefox tests/bench_result/index.html &
