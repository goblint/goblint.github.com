How to reproduce benchmarking results?

  1. Install & configure Ocaml (≥ 3.12.1) and Ruby 
    ???
  2. Create a directory, where we will install Goblint and some required libraries. 
    mkdir goblint_stuff
cd goblint_stuff
  3. Download and prepare the xml-light library: 
    cvs -d:pserver:anonymous@cvs.motion-twin.com:/cvsroot login
(enter empty password)
cvs -d:pserver:anonymous@cvs.motion-twin.com:/cvsroot co ocaml/xml-light
mv ocaml/xml-light xml-light
rm -rf ocaml
cd xml-light
make
make opt
cd ..
  4. Download and prepare the cil library: 
    git clone git://github.com/kerneis/cil
cd cil
./configure
make
cd ..
  5. Download our benchmarking code: 
    git clone git://github.com/goblint/bench.git
  6. Download and prepare the Goblint: 
    git clone git://github.com/goblint/analyzer.git:
git checkout 8397e45088e4ce88f6c75d871b02e0268af0ee21
cd analyzer/
./configure
make opt
  7. Run & Review results: 
    ./scripts/update_bench_side.rb
