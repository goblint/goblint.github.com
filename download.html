Download and Installation

Since the analyzer is rapidly developing, we recommend using the development versions of Goblint and CIl as described on our developer pages on GitHub.

Requirements

  1. Ocaml >=4.00.1
  2. Findlib (tested with 1.3.3)
  3. Camomile (tested with 0.8.3)
  4. Batteries >=2.0
  5. Xml-light 2.2
  6. CIL (latest from git)
Batteries, xml-light, and CIL must be accessible using ocamlfind. Consult the INSTALL file for more detailed instructions. You can test if the analyzer works by running it on the test programs; for example: 
   ./goblint tests/regression/04-mutex/01-simple_rc.c
If this works, you are ready to use the analyzer.
