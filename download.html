Download and Installation

Since the analyzer is rapidly developing, we recommend using the development versions of Goblint and CIL. For Windows, you may use a custom cygwin repository that installs cygwin and goblint.

Binary Distribution (Windows)

Currently, we only provide binary builds on Windows based on the Cygwin environment.
  1. Download the following archive: goblint.zip
  2. Extract the archive and run the "install" batch file.
  3. Click "Next", "Next", "Next", .... (the goblint package is preselected).
  4. Open the cygwin shell and test goblint: goblint /usr/share/goblint/tests/01-simple_rc.c

Building from Sources

Please see the readme file on our github pages for build instructions. There is also prebuilt virtual machine images to test the analyzer.
